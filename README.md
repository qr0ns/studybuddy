# 🤖 StudyBuddy — обучающий Telegram-бот

StudyBuddy — это Telegram-бот, который помогает пользователям решать учебные задачи и находить разборы.

## 🚀 Возможности

* Выбор предмета (математика, информатика, физика)
* Выбор задания через inline-клавиатуру
* Поиск видеоразборов (YouTube)
* Генерация ответов с помощью нейросети (OpenAI API)

## 🛠 Технологии

* Python
* aiogram
* OpenAI API
* Telegram Bot API

## 📸 Пример работы

Бот предлагает выбрать предмет → задание → способ решения:

* получить видеоразбор
* получить ответ от нейросети

## ⚙️ Установка

```bash
git clone https://github.com/qr0ns/studybuddy
cd studybuddy
pip install -r requirements.txt
cd app
python main.py
```

## 🔑 Переменные окружения

```
BOT_TOKEN=your_token
OPENAI_API_KEY=your_key
```
